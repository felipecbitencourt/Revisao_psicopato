Demonstração do A11y Kit

Clique no botão flutuante (canto inferior direito) para abrir o painel. Arraste-o para qualquer canto. Tudo é persistido no localStorage.

Por que linguagem simples importa

As instituições utilizam, com frequência, termos que dificultam a compreensão. A fim de ampliar o acesso, é necessário solicitar revisões e demonstrar que o texto pode ser mais direto.

Este parágrafo possui palavras formais propositais para a demonstração do simplificador: utilizar, solicitar, mediante, posteriormente.

Leitura assistiva

  • Ative a leitura biônica para destacar o início das palavras.
  • Use a fonte Atkinson ou Lexend na aba Avançado.
  • Experimente os presets de Assistência de leitura.
